# Talk to Prompt

This is a plain HTML file, that can be opened in any browser. 

Instead of typing a lot to create Prompt, just talk to it, and it will cleanse and format spoken words into a well structured Prompt. 

The code assumes Azure OpenAI models, adjust variables to point to your model deployments to be used for this.

If you are using another LLM, you will need to adjust calls in JavaScript code for that.
